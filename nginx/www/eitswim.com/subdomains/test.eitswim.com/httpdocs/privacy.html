個人情報保護方針

株式会社Agent Blue（以下、弊社）は、お客様の個人情報を保護し、プライバシーを尊重するために、弊社が個人情報保護についての重要性を認識し、以下の取り組みを周知徹底させ実施いたします。

  1. 弊社は、お客様の個人情報の収集にあたっては、適法かつ適正な方法で行います。
  2. 弊社は、お客様の個人情報の利用にあたっては、その利用目的を明確にすると共に、適切な利用および提供を行います。
  3. 弊社は、お客様の個人情報の管理、ならびに取り扱いにあたっては、個人情報の漏洩、減失または、棄損の防止、その他の安全管理のために必要かつ適切な措置を講じます。
  4. 弊社は、お客様の個人情報について、開示や訂正等、弊社所定の手続きに基づき、適切な範囲で対応します。
  5. 弊社は、個人情報管理責任者を任命し、個人情報を管理する管理者を置き、お客様からお預かりする個人情報の取り扱いについて適切な管理を行います。
  6. 弊社は、個人情報について適用される法令、その他の規範を遵守します。

個人情報の取り扱いについて

  1. 個人情報の利用目的について
    弊社は、個人情報を弊社が行う下記業務において下記利用目的の達成に必要な範囲で利用します。
  2. 個人情報の定義
    個人情報とは、個人情報の保護に関する法律に規定される生存する個人に関する情報（氏名、生年月日、その他の特定の個人を識別することができる情報）、ならびに特定の個人と結びついて使用されるメールアドレス、ユーザーＩＤ、パスワードなどの情報、および個人情報と一体となった趣味、家族構成、年齢その他の個人に関する属性情報であると認識しています。
  3. 個人情報利用目的の特定
    弊社は、個人情報を取り扱うにあたって、その利用の目的を出来る限り特定します。
  4. 個人情報利用の制限
    弊社は、あらかじめご本人の同意を得ず、利用目的の達成に必要な範囲を超えて個人情報を取扱うことはありません。合併その他の理由により個人情報を取得した場合にも、あらかじめご本人の同意を得ないで、承継前の利用目的の範囲を超えて取扱うことはありません。ただし、次の場合はこの限りではありません。
    1. 法令に基づく場合
    2. 人の生命、身体または財産の保護のために必要がある場合であって、ご本人の同意を得ることが困難であるとき
    3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、ご本人の同意を得ることが困難であるとき
    4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、ご本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき
  5. 個人情報の取得
    ユーザーは、弊社の提供するサービスの利用及び弊社とのコミュニケーションに際しては、名前、住所、電話番号、メールアドレス等の個人を特定し得る情報（以下これらの情報を総称して「個人情報」といいます。）の提供を求める場合があります。弊社の提供するサービスの利用に関連して、弊社では、その他の情報、即ちブラウザ情報及びスマートフォンアプリ情報等で取得された情報、ピクセルタグその他の技術情報、並びに人口統計的情報その他の集合情報等の個人を特定しない情報（以下かかる個人を特定しない情報を総称して「非個人情報」といいます。）も取得する場合があります。弊社は、個人情報を本プライバシーポリシーに従って取り扱います。弊社は、かかる個人情報を本プライバシーポリシーに定める目的のために利用することができます。
    1. 個人情報
      弊社は、以下の場合に情報提供者の個人情報を取得します。
      1. 弊社の提供するサービスに入会又は登録する場合
      2. 弊社の提供するサービス内でマイページを作成・編集する場合
      3. 弊社の提供するサービス内の機能を利用する際に、情報提供者が任意に情報を入力する場合（ボタンのクリック、アプリケーションのダウンロード又はリンクを開く行為を含みます）
      4. 弊社のサポートサービスに対して連絡をした場合
      5. その他、情報提供者の明確な同意が得られた場合
    2. 非個人情報を取得する場合
      弊社は、以下の場合に情報提供者の非個人情報を取得します。
      1. 情報提供者のインターネットブラウザにあらかじめ設定されている頻度で、インターネットブラウザの履歴情報フォルダから情報を取得する場合。
      2. ウェブブラウザのクッキーから情報を取得する場合。
        「クッキー」とは、文字列が記入された小さいデータファイルのことで、弊社のサービスが提供されているウェブサイトから、当該ウェブサイトにアクセスした情報提供者のブラウザに送信され、そのブラウザによって情報提供者のコンピューターのハードドライブ等の記憶装置に保存されます。情報提供者のシステムにクッキーが保存されることで、弊社は簡便に弊社の提供するサービスをユーザーに合わせてカスタマイズし、弊社の提供するサービスの利便性を向上させることができます。また、この機能は、弊社が問題を検証し、弊社の提供するサービスを管理するためにも役立ちます。弊社は、クッキーを利用して、第三者から情報提供者のデータを受け取る場合があります。クッキーその他の技術によって取得された情報の一部は、広告表示の最適化のために利用されます。広告表示の最適化のためのクッキーの無効化を希望される方は、クッキーの無効化に関する設定から行うことができます。弊社では、情報提供者の責任かつ適切な方式の範囲内においてクッキーを有効にすることを推奨していますが、クッキーを無効にすると、弊社の提供するサービスの一部の機能が正常に機能しなくなることがあります。
      3. ピクセルタグ、ウェブビーコン、クリアGIFその他の類似技術により取得する場合。
        弊社は、弊社の提供するサービス及び弊社が発行するHTML形式のメールに関連して、とりわけユーザーやメール受領者の反応履歴を確認するために、これらの技術を利用する場合があります。これには、弊社のマーケティング活動の成果を評価する場合並びに使用率及び回答率に関する統計を作成する場合を含みます。
      4. 情報を集合化する場合。
        例えば、弊社は集合化した個人情報を利用して、特定の市外局番を持つユーザーの割合を算出する場合があります。かかる集合化した個人情報によって、個人又はその他のユーザーが特定されることはありません。

      非個人情報では個人を特定することは不可能であるため、弊社は非個人情報を使用し又は開示する場合があります。場合により、弊社は非個人情報と個人情報とを組み合わせる場合があります。弊社が非個人情報と個人情報とを組み合わせた場合、組み合わされた情報は組み合わされている間は、個人情報として取扱われます。

    3. その他の取得情報
      以下は、ユーザーが弊社が提供するサービスを利用することに関連して、弊社が取得する場合があるその他の個人情報及び非個人情報の例です。
      1. ユーザーが新しい端末からログインした場合：過去の端末ID、現在の端末ID及びログインの日時
      2. ユーザーが弊社の提供するサービスを利用する場合：友人への招待メール、日記、コミュニティ、メッセージボード、プロフィールの写真その他の写真、ユーザーのメール及びユーザーの短いコメントに含まれるか又はそれらにおいて利用可能な情報
      3. ユーザーが弊社の提供するサービスを利用する場合：IPアドレス、使っている端末の種類、端末の言語設定、ブラウザーの種類、アプリのバージョン
      4. IPアドレスは、インターネットサービスプロバイダー（ISP）から、ユーザーが使用しているコンピューターに自動的に割り当てられる数字です。弊社は、ユーザーが弊社の提供するサービスにアクセスしたときは常に、アクセスの時間及びアクセスしたウェブページに加えて、自動的に、弊社のサーバーログファイル内にログされ、特定されるIPアドレスを取得する場合があります。弊社は、適用される法により別段の扱いを行なう義務がない場合、IPアドレス、サーバーログファイル及び関連する情報を非個人情報として扱います。弊社が本条項に記載された用途に加えて、IPアドレスを個人情報として扱う必要がある場合、弊社は、IPアドレスを、弊社が個人情報を使用し又は開示する全ての目的のために使用し又は開示する場合があります。
      5. ユーザーが弊社の提供するサービスを経由して支払いを行った場合：選択した決済手段、決済毎に付与される番号、ユーザーが支払い時に決済代行会社に提供するクレジットカード並びに銀行口座番号、パスワード、郵便番号、住所、氏名、メールアドレス及び電信送金の情報その他の情報。これらの情報の取得は、決済処理を行うことを目的としています。
      6. 弊社のサービスに登録した場合：名前及び連絡先
        弊社のウェブサイト及びスマートフォンアプリにアクセスした場合：当該アクセスをした者のアクセスログ。
        ユーザー登録をしたユーザーが、弊社のウェブサイトやスマートフォンアプリにアクセスした場合：弊社サービスへのアクセスログや履歴情報、ユーザーが利用されたサービス、ソフトウエア、購入された商品、ご覧になったページや広告の履歴情報、利用時間帯、利用の方法、利用環境（利用に際しての各種設定情報なども含みますがこれに限られません）、ユーザーのIPアドレス、cookie情報、位置情報、端末の個体識別情報などの履歴情報及び特性情報を、弊社のサービスの品質向上やユーザーの嗜好性把握、広告表示の最適化の目的で、弊社または弊社と秘密保持契約を結んだ第三者（情報提供元、広告主、広告配信先（Supply-side Platform, Demand-side Platformなどを含みます。）が収集し、共有する場合があることを、予めご承諾頂きます 。第三者は弊社サービス以外のページにて情報を取得することがあり、弊社は第三者が取得した情報をユーザーのクッキー情報と照合、ひもづけて共有することによって、さらなる広告表示の最適化を行います。
  6. 個人情報利用目的の変更
    弊社は、個人情報の利用目的を変更する場合には、変更前の利用目的と相当の関連性を有すると合理的に認められる範囲を超えては行わず、変更された利用目的について、ご本人に通知し、または公表します。
  7. 個人情報の安全管理・従業員の監督
    弊社は、個人情報の漏洩、滅失またはき損の防止その他の個人情報の安全管理が図られるよう、個人情報保護規程を定め、従業員に対する必要かつ適切な監督を行います。
  8. 第三者提供の制限
    弊社は、次に掲げる場合を除くほか、あらかじめご本人の同意を得ないで、個人情報を第三者に提供しません。
    1. 法令に基づく場合
    2. 人の生命、身体または財産の保護のために必要がある場合であって、ご本人の同意を得ることが困難であるとき
    3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、ご本人の同意を得ることが困難であるとき
    4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、ご本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき
    5. 予め次の事項を告知あるいは公表をしている場合
      1. 利用目的に第三者への提供を含むこと
      2. 第三者に提供されるデータの項目
      3. 第三者への提供の手段または方法
      4. ご本人の求めに応じて個人情報の第三者への提供を停止すること

      ただし次に掲げる場合は上記に定める第三者には該当しません。

      1. 弊社が利用目的の達成に必要な範囲内において個人情報の取扱いの全部または一部を委託する場合
      2. 合併その他の事由による事業の承継に伴って個人情報が提供される場合
      3. 個人情報を特定の者との間で共同して利用する場合であって、その旨並びに共同して利用される個人情報の項目、共同して利用する者の範囲、利用する者の利用目的および当該個人情報の管理について責任を有する者の氏名または名称について、あらかじめご本人に通知し、またはご本人が容易に知り得る状態に置いているとき

      弊社は、個人情報を特定の者との間で共同して利用する場合、利用目的または管理責任者の氏名または名称が変更される場合は、変更する内容について、あらかじめご本人に通知し、またはご本人が容易に知り得る状態に置きます。

  9. 個人情報に関する事項の公表等
    弊社は、個人情報に関する次に掲げる事項について、ご本人の知り得る状態に置き、ご本人の求めに応じて遅滞なく回答します。
    1. 個人情報の利用目的（ただし、個人情報の保護に関する法律において、その義務がないと規定されるものは除きます。ご回答しない決定をした場合は、ご本人に対して遅滞なくその旨を通知します。）
    2. 個人情報に関するお問い合わせ窓口
  10. 個人情報の開示
    弊社は、ご本人から、個人情報の開示を求められたときは、ご本人に対し、遅滞なく開示します。ただし、開示することにより次のいずれかに該当する場合は、その全部または一部を開示しないこともあり、開示しない決定をした場合には、その旨を遅滞なく通知します。
    1. ご本人または第三者の生命、身体、財産その他の権利利益を害するおそれがある場合
    2. 弊社の業務の適正な実施に著しい支障を及ぼすおそれがある場合
    3. 他の法令に違反することとなる場合

    なお、アクセスログなどの個人情報以外の情報については、原則として開示いたしません。

  11. 個人情報の訂正等
    弊社は、ご本人から、個人情報が真実でないという理由によって、内容の訂正、追加または削除（以下「訂正等」といいます）を求められた場合には、他の法令の規定により特別の手続きが定められている場合を除き、利用目的の達成に必要な範囲内において、遅滞なく必要な調査を行い、その結果に基づき、個人情報の内容の訂正等を行い、その旨ご本人に通知します。
  12. 個人情報の利用停止等
    弊社は、ご本人から、ご本人の個人情報が、あらかじめ公表された利用目的の範囲を超えて取り扱われているという理由、または偽りその他不正の手段により取得されたものであるという理由により、その利用の停止または消去（以下「利用停止等」といいます）を求められた場合には、遅滞なく必要な調査を行い、その結果に基づき、個人情報の利用停止等を行い、その旨ご本人に通知します。ただし、個人情報の利用停止等に多額の費用を有する場合その他利用停止等を行うことが困難な場合であって、ご本人の権利利益を保護するために必要なこれに代わるべき措置をとれる場合は、この代替策を講じます。
  13. 理由の説明
    弊社は、ご本人からの要求にもかかわらず、
    1. 利用目的を通知しない
    2. 個人情報の全部または一部を開示しない
    3. 個人情報の利用停止等を行わない
    4. 個人情報の第三者提供を停止しない

    のいずれかを決定する場合、その旨ご本人に通知する際に理由を説明するよう努めます。

  14. 免責事項
    個人情報の漏洩等がないよう当社は法令および当社規程に基づき適切にサービスの運営を行いますが、以下の場合において第三者が個人情報を取得したことにより発生したトラブルについて免責され何らの法的責任も負わないものとします。
    1. 利用者自らが当社のサービス上の機能または、別の手段を用いて他の利用者に個人情報を明らかにした場合
    2. 利用者本人以外の利用者が当社のサービス上で入力した情報により、ご本人が特定された場合
    3. 当社が提供するサービス上の広告サイトを利用者が利用したことに起因して発生した漏洩等の事故の場合
    4. その他当社に過失がないにも拘わらず発生した漏洩等の事故の場合
